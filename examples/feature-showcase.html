Required Files

Include these files in your HTML <head> section:

<link rel="stylesheet" href="isometric-3d.css">
<script src="isometric-3d.js"></script>

The library automatically initializes all .isometric-perspective elements on page load. No additional JavaScript is needed for basic functionality!

Note: The library automatically adds the .face class to all cuboid faces (front, back, left, right, top, bottom). Use .cuboid .face to style all faces, or target specific faces like .cuboid .front for individual styling.

Flat 2D Scenes

What is it?

A flat scene is a simple 2D element positioned in 3D space without cube faces. Perfect for labels, titles, or simple content.

  • Uses data-z-axis for vertical positioning
  • No data-width, data-height, or data-depth needed
  • Content flows naturally like a regular div

Interactive Navigation

What is it?

Click on faces to navigate to predefined camera positions. Perfect for guided presentations and storytelling.

  • data-nav-xyz sets target rotation (format: "x.y.z")
  • data-nav-zoom sets target zoom level
  • data-section creates navigation points in the nav bar
  • Smooth animated transitions between views

     data-width="120" 
     data-height="80"

  <div class="right">Right</div>

       data-nav-zoom="1.3"

    üìç Click: Top View

       data-nav-xyz="-45.0.-35"

       data-section="bottom-view">

</div>

Highlighting and Focus System

What is it?

Dynamically highlight specific elements while dimming others. Perfect for step-by-step presentations and guided tours. This example uses a 3√ó3 grid layout with connector vertices for curved paths.

  • data-groups assigns elements to named groups
  • data-activate on navigation triggers highlighting
  • Highlighted elements receive .highlight class
  • Non-highlighted elements receive .dimmed class and automatically dim
  • Works with cuboids, faces, and SVG connectors
  • Vertices: Control connector path curves using "vertices": "20,80" values (0-100, where 50 is center)

Customizing Dimming Effect

Adjust the dimming intensity by modifying the CSS opacity values in isometric-3d.css:

/* Dim connectors (default: 0.1) */
.connector-path.dimmed {
  opacity: 0.1;
}

/* Dim scenes (default: 0.2) */
.scene.dimmed {
  opacity: 0.2;
}

/* Dim cuboid faces (default: 0.2) */
.cuboid.dimmed > .face,
.face.dimmed {
  opacity: 0.2;
}

/* Dim cuboid content (default: 0.4) */
.cuboid.dimmed > *:not(.face) {
  opacity: 0.4;
}
HTML 
<div class="isometric-perspective"
     data-connectors='[{
  "ids": "cube1,cube2",
  "positions": "center,center",
  "vertices": "35,65",
  "color": "#221F19",
  "groups": "step1"
}]'>
  <div id="cube1" 
       class="cuboid"
       data-groups="step1"
       data-width="60"
       data-height="50"
       data-depth="60">
    <div class="front" 
         data-groups="step1">
      Start
    </div>

      üìç Step 1

  </div>
  <!-- more cuboids... -->
</div>

Automatic Height Calculation

What is it?

When data-height is omitted, the system automatically calculates the height based on the face content. Perfect for dynamic content.

  • Omit data-height attribute to enable auto-calculation
  • System measures actual content height of side faces
  • Great for flex layouts with variable items
  • Content determines dimensions automatically
