Key Differences from Cuboid Examples:

  • ‚úÖ No Cuboids: Only scene elements (no 3D boxes with faces)
  • ‚úÖ Direct Flex Layout: Scenes use flexbox directly for layout
  • ‚úÖ Simpler Structure: Cleaner HTML without cuboid/face hierarchy
  • ‚úÖ Shadows Still Work: Scenes with data-z-axis cast shadows
  • ‚úÖ Navigation: Click scenes to navigate and highlight groups

Features:

  • ‚úÖ Nested Scenes: Scene A contains A1 & A2, Scene D contains D1, D2 & D3
  • ‚úÖ Flex Column Layout: All scenes stacked vertically using .flex-column
  • ‚úÖ Color Coding: Each scene family has its own color theme
  • ‚úÖ Elevation: data-z-axis creates depth and shadows
  • ‚úÖ Highlight Groups: Each scene has a data-groups attribute

What to try:

  • Click Scene A1 or A2 - Highlights the entire Scene A group (blue theme)
  • Click Scene B - Highlights Scene B (green)
  • Click Scene C - Highlights Scene C (purple)
  • Click Scene D1, D2, or D3 - Highlights the entire Scene D group (orange theme)
  • Drag and rotate - See how scenes maintain their flex layout in 3D
  • Use compact controls - Hover bottom-right for navigation sphere

Technical Details:

  • .flex-column - Stacks child scenes vertically with gap
  • data-z-axis - Elevates scenes in 3D space and creates shadows
  • data-groups - Groups scenes for coordinated highlighting
  • data-activate - Triggers highlight for specified group
  • data-nav-xyz - Sets camera rotation on click
  • data-nav-zoom - Sets zoom level on click
  • No data-width, data-height, data-depth - These are cuboid-only attributes

Simple nested scene pattern:

üé® Customization Guide (See CSS Comments Above):

  • Container Background: Change .isometric-container background for different atmospheres (gradients, solids, patterns)
  • Scene Colors: Customize individual scene backgrounds by ID (#sceneA, #sceneB, etc.)
  • Border Styles: Modify .scene border width, color, radius for different looks
  • Glass Effect: Adjust backdrop-filter: blur() for more/less transparency
  • Hover Scale: Change --scene-hover-scale value (1.0 = no scale, 1.1 = 10% larger)
  • Shadows: Adjust box-shadow for depth perception
  • Typography: Customize font-family, size, weight, color, and text-shadow
  • Animations: Uncomment the animation examples at the bottom of CSS for entrance effects
  • Responsive: Add media queries for mobile/tablet optimizations

üí° CSS Variables You Can Control:
