✅ Migration Complete!

This example demonstrates the new structure:

  • ✅ .scene elements are now containers (no dimensions)
  • ✅ .cuboid elements have dimensions (data-width, data-height, data-depth)
  • ✅ Face elements use direction classes only (.front, .back, .left, .right, .top, .bottom)
  • ✅ No more .face class needed
  • ✅ Shadows still attach to .scene elements via data-z-axis
  • ✅ Better semantic structure: scenes contain cuboids and other scenes

Structure:

Scene A (container)
  ├─ Scene A1 (data-z-axis="20" - has shadow)
  │   └─ Cuboid (80×60×50)
  │       └─ 6 faces
  └─ Scene A2 (data-z-axis="20" - has shadow)
      └─ Cuboid (80×60×50)
          └─ 6 faces

Scene B (data-z-axis="50" - has shadow)
  └─ Cuboid (100×120×100)
      └─ 6 faces

Scene C (data-z-axis="10" - has shadow)
  └─ Cuboid (100×auto×80)
      └─ 6 faces (front has 3 sub-elements)
